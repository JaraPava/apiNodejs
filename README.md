#PROYECTO API NODE.JS

==== REQUISITOS PARA PODER LEVANTAR ESTA APLICACIÓN: ====
```
    1. Instalar node.js:
         https://nodejs.org/en
    2. Instalar mongodbCompass:
        https://www.mongodb.com/products/tools/compass

    NOTA: Se esta trabajando con las últimas versiones.
    También para saber si se instaló correctamente el node.js, se puede ingresar
    el siguiente comando en consola: node --version y este seguidamente te
    mostrará la versión e instalado correctamente.
```

==== INSTALACIÓN Y EJECUCIÓN DE LA APLICACIÓN  ====
```
    1. Abrimos la terminal desde la raiz del proyecto
    2. Ingresamos el siguiente comando: npm i
    3. Después ejecutamos el siguiente comando: npm run dev.
```

==== ¿COMO USAR LA APLICACIÓN ====
```
    1. Abrimos el app.ts que esta ubicado en el src
    2. Para poder cerrar las regiones basta con realizar el siguiente atajo: ctrl + k + 0
    3. Abrimos la función async function main()
    4. Se mostrará una lista de regiones, favor de leer el titulo de cada región, por ejemplo
    //#region get all products; que a su vez y por lógica infiere que traera todos los productos
    de la base de datos registrados por los usuarios.
    5. Descomentamos las líneas de código que estan entre esas regiones
    6. Guardamos cambios
    7. Se mostrará por consola la información al ejecutar cada acción.
    NOTA: Para el siguiente commit se mostrará y se implementarán de mejor manera las peticiones y las respuestas.Al igual que las validaciones
    De igual manera es muy importante que se use solamente una región a la vez. Aun no tiene estabilidad
```
